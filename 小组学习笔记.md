## 二、图论和多智能体系统一致性

### 1.图论用处及基础概念

- 描述谁和谁通信，通信的结构
- 图是点加边：点（智能体）边（谁和谁通信），写成G=(V,E)
- 顶点：几何上是空间中一个点，是图的构成，但是在智能体中的理解是代表一个独立的智能体，拥有自己的状态变量
- 邻居：几何上是通过一条边紧挨的点，智能体上决定信息传递的范围
- 度：可类比为影响力，程度；有入度和出度之分，入度是被指向的数量，出度反之

### 2.图有什么

- 无向图：边是双向的，向微信好友之间的关系，彼此是好友
- 有向图：边是单向的，像我喜欢你但是你不喜欢我
- 加权图：边有权重，就像导航，可以看到ab之间是差距十分钟
- 非加权图：边无权重，像微信好友一样
- 简单图：没有自环和重复边
- 复杂图;允许自环和重复边（如城市的线路布局）

### 3.图的存储方式

- 邻接矩阵：可理解为 j可以给i发信息，那么$A_{ij}$=1,反之等于0
  - 缺点是点多很浪费时间
- 邻接表：列出每个节点能和谁连
  - 省空间，写python必须用邻接表画拓扑图
- 边表：直接把边一条一条列出来
- 关联矩阵 

### 4.图的基础操作

- 操作1：查询两个节点是否相邻，判断两个智能体有没有通信
- 操作2：求一个节点的度
- 操作3：添加一条边
  - 无向图加边 $A_[ij]$=1,$$A_{ji}$$=1
  - 有向图加边 $A_{ij}$=1
- 操作4：删边，前面1变为0即可
- 操作5：遍历所有邻居（广度优先遍历）一层一层 往外扩散，用队列实现。

### 5.基础操作中一些概念

- 1.路径：从一个节点到另外一个节点序列，相邻节点之间有边
- 2.简单路径：路径节点不重复
- 3.回路：起点和终点是同一个节点的路径
- 4.连通：无向图为任意两点都有路径，有向图为任意两点之间互相有路径可达
- 5.连通分支：无向图中互不连通的极大子图

### 6。遍历

- 判断图连通性，找连通分支、判环、找最短路径

- | 变量        | 含义               | 初始值     |
  | ----------- | ------------------ | ---------- |
  | `visited[]` | 标记节点是否被访问 | 全 `False` |

- DFS：优先深度探索，主要是一条路走到底
  - 步骤：
    - 1.访问当前节点 u，标记 visited[u] = True
      2.遍历 u 的所有邻居 v
      3.若 v 未访问，递归执行 DFS(v)
      4.所有邻居遍历完自动回溯

- BFS：队列式 “一层一层扩散”，优先访问近节点

  - 步骤：

    - 1.起点入队，标记 visited[start] = True
      2.队列非空时，取出队头节点 now 并访问
      3.遍历 now 的所有邻居 v
      4.若 v 未访问，标记后入队
      5.重复至队空

    主要可以最短路径

| 维度     | DFS                | BFS            |
| -------- | ------------------ | -------------- |
| 数据结构 | 递归 / 栈          | 队列           |
| 访问顺序 | 深度优先           | 广度优先       |
| 核心优势 | 空间开销小、易实现 | 能找最短路径   |
| 典型场景 | 判环、找连通分支   | 无权图最短路径 |

```python
# 1. 定义图（邻接表形式，节点编号从0开始）
graph = [
    [1, 3],   # 节点0的邻居：1、3
    [0, 2],   # 节点1的邻居：0、2
    [1, 3],   # 节点2的邻居：1、3
    [0, 2]    # 节点3的邻居：0、2
]

# 全局访问标记（初始化全为False）
visited = [False] * len(graph)

# 2. DFS实现(递归)
def dfs(u):
    """
    u: 当前访问的节点编号
    """
    print(f"访问节点: {u}", end=" ")
    visited[u] = True  # 标记已访问
    # 遍历当前节点的所有邻居
    for v in graph[u]:
        if not visited[v]:  # 邻居未访问则递归
            dfs(v)

# 3. BFS实现
from collections import deque  
def bfs(start):
    """
    start: 遍历起点
    """
    # 重置访问标记
    visited_bfs = [False] * len(graph)
    queue = deque()
    queue.append(start)       # 起点入队
    visited_bfs[start] = True # 标记已访问
    
    while queue:  # 队列非空时循环
        now = queue.popleft() # 取出队头
        print(f"访问节点: {now}", end=" ")
        # 遍历当前节点的所有邻居
        for v in graph[now]:
            if not visited_bfs[v]:
                visited_bfs[v] = True
                queue.append(v)

if __name__ == "__main__":
    print("=== DFS遍历结果（起点0）===")
    dfs(0)  # 从节点0开始DFS
    
    print("\n=== BFS遍历结果（起点0）===")
    bfs(0)  # 从节点0开始BFS

  
    print("\n=== 判断图是否连通 ===")
    # 重置标记
    visited_check = [False] * len(graph)
    dfs(0)  # 一次DFS遍历
    if all(visited_check):
        print("图是连通的")
    else:
        print("图不连通")
```

### 7.图的连通性和生成树

- 

- | 知识点       | 核心定义                                                     | 与一致性的关联                                         |
  | ------------ | ------------------------------------------------------------ | ------------------------------------------------------ |
  | 有向图连通性 | 弱连通：忽略方向后连通；强连通：任意两点互相可达；有向生成树：存在根节点能到达所有其他节点 | 多智能体一致性的核心拓扑条件：通信图包含**有向生成树** |
  | 生成树       | 连通图的极小连通子图（n 个节点 n-1 条边，无环）              | 一致性算法中，生成树是信息传递的最小拓扑               |
  | 代数连通度   | 无向图拉普拉斯矩阵的第二小特征值 λ₂                          | λ₂>0 ↔ 图连通；λ₂越大，一致性收敛越快                  |

- 三大矩阵

| 矩阵           | 定义                                                     | 关键性质                                                 |
| -------------- | -------------------------------------------------------- | -------------------------------------------------------- |
| 邻接矩阵 A     | A$_{ij}$=1（i→j 有边），否则 0                           | 无向图 A 对称；有向图 A 不一定对称                       |
| 度矩阵 D       | 对角矩阵，D$_{i}$= 节点 i 的度（无向图）/ 出度（有向图） | 仅对角线非 0，其余为 0                                   |
| 拉普拉斯矩阵 L | L = D - A                                                | 无向图 L 对称半正定；行和为 0；λ₁=0（对应全 1 特征向量） |

- 多智能体一阶一致性算法公式
- 

| 算法类型           | 公式                                                         | 收敛条件                             |
| ------------------ | ------------------------------------------------------------ | ------------------------------------ |
| 连续时间一阶一致性 | $\dot{x}_i = \sum_{j\in\mathcal{N}_i}(x_j - x_i)$   矩阵形式：   $\dot{\boldsymbol{x}} = -\boldsymbol{L}\boldsymbol{x}$ | 无向图连通 / 有向图含生成树          |
| 离散时间一阶一致性 | $x_i(k+1) = x_i(k) + \varepsilon\sum_{j\in\mathcal{N}_i}(x_j - x_i)$   矩阵形式：   $\boldsymbol{x}(k+1) = (\boldsymbol{I} - \varepsilon\boldsymbol{L})\boldsymbol{x}(k)$ | 0<ε<2/λ_${n}$（λ_n 为 L 最大特征值） |

### 四。json库，作用：

   - 将python和json来回转换

   - 类型转换

- | Python 类型      | JSON 类型 | 你的应用场景                                   |

- 四个目前学的操作

  - 保存仿真配置到json文件：

       - 1.open(文件名，“w”，encoding=“utf-8”):以写入模式打开文件，指定编码避免中文乱码

       - 2.json.dump(数据，文件对象，indent=4,ensure_ascii=False):把python数据写入json文件

       - 3.numpy数组要先转为列表（.tolist()),因为json不支持np.array类型

```python
import json
import numpy as np
import math
import os


class 坐标系:
    """简单的坐标系类"""

    def __init__(self, 坐标轴, 名称="当前坐标系"):
        """
        初始化坐标系

        参数:
        坐标轴: 比如 [[1,0], [0,1]] 表示二维直角坐标系
        名称: 坐标系的名字
        """
        self.坐标轴 = np.array(坐标轴, dtype=float)
        self.名称 = 名称
        self.维度 = self.坐标轴.shape[0]  # 看看是几维的
        self.向量列表 = []  # 存放这个坐标系里的向量

        # 检查坐标系是否有效
        print(f"\n检查坐标系 {名称}:")
        if not self.检查是否有效():
            print(f"警告：{名称} 的坐标轴可能有问题！")

    def 检查是否有效(self):
        """检查坐标轴是否线性独立（行列式不为0）"""
        try:
            行列式 = np.linalg.det(self.坐标轴)
            print(f"  行列式 = {行列式:.6f}")

            if abs(行列式) < 1e-10:
                print(f"  行列式接近于0，坐标轴可能线性相关")
                return False
            else:
                print(f"  坐标系有效")
                return True
        except:
            print(f"  计算行列式时出错")
            return False

    def 添加向量(self, 向量, 名字=None):
        """向坐标系里添加一个向量"""
        向量信息 = {
            '向量': np.array(向量, dtype=float),
            '名字': 名字 or f"向量{len(self.向量列表)}"
        }
        self.向量列表.append(向量信息)

    def 坐标系转移(self, 原向量, 目标坐标系):
        """
        把向量从一个坐标系转到另一个坐标系
        """
        print(f"\n--- 坐标系转移 ---")
        print(f"原向量: {原向量}")

        # 第1步：把原向量变成numpy数组
        原向量 = np.array(原向量, dtype=float)

        # 第2步：先把向量转到标准直角坐标系
        原坐标轴矩阵 = self.坐标轴
        v_直角坐标系 = np.dot(原坐标轴矩阵, 原向量)
        print(f"在直角坐标系中: {v_直角坐标系}")

        # 第3步：再从直角坐标系转到目标坐标系
        目标坐标轴矩阵 = 目标坐标系.坐标轴
        v_目标坐标系 = np.linalg.solve(目标坐标轴矩阵, v_直角坐标系)
        print(f"在目标坐标系中: {v_目标坐标系}")

        return v_目标坐标系

    def 投影长度(self, 向量):
        """
        计算向量在每个坐标轴上的投影长度
        """
        print(f"\n--- 计算投影长度 ---")
        print(f"向量: {向量}")

        向量 = np.array(向量, dtype=float)

        # 先把向量转到直角坐标系
        v_直角 = np.dot(self.坐标轴, 向量)
        print(f"在直角坐标系中: {v_直角}")

        # 计算在每个轴上的投影
        投影结果 = []
        v_长度 = np.linalg.norm(v_直角)
        print(f"向量长度: {v_长度:.4f}")

        for i in range(self.维度):
            # 取出第i个坐标轴
            轴 = self.坐标轴[:, i]
            轴长度 = np.linalg.norm(轴)

            # 投影长度 = |v| * cosθ = (v·轴) / |轴|
            点积 = np.dot(v_直角, 轴)
            投影 = abs(点积) / 轴长度

            投影结果.append(投影)
            print(f"  在轴{i + 1}上的投影: {投影:.4f}")

        return 投影结果

    def 夹角弧度(self, 向量):
        """
        计算向量与每个坐标轴的夹角（弧度）
        """
        print(f"\n--- 计算夹角 ---")
        print(f"向量: {向量}")

        向量 = np.array(向量, dtype=float)

        # 转到直角坐标系
        v_直角 = np.dot(self.坐标轴, 向量)
        v_长度 = np.linalg.norm(v_直角)

        print(f"在直角坐标系中: {v_直角}")
        print(f"向量长度: {v_长度:.4f}")

        if v_长度 == 0:
            print("向量是零向量，无法计算夹角")
            return [0] * self.维度

        夹角结果 = []

        for i in range(self.维度):
            轴 = self.坐标轴[:, i]
            轴长度 = np.linalg.norm(轴)

            # 计算cosθ = (v·轴) / (|v| * |轴|)
            点积 = np.dot(v_直角, 轴)
            cos_theta = 点积 / (v_长度 * 轴长度)

            # 防止计算误差导致超出[-1,1]
            cos_theta = max(-1, min(1, cos_theta))

            # 用反余弦求角度
            角度 = math.acos(cos_theta)
            角度_度 = math.degrees(角度)

            夹角结果.append(角度)
            print(f"  与轴{i + 1}的夹角: {角度:.4f} 弧度 ({角度_度:.2f}°)")

        return 夹角结果

    def 面积缩放(self):
        """
        计算这个坐标系相比直角坐标系，面积放大了多少倍
        """
        print(f"\n--- 计算面积缩放 ---")

        行列式 = np.linalg.det(self.坐标轴)
        缩放倍数 = abs(行列式)

        print(f"坐标轴矩阵的行列式 = {行列式:.6f}")
        print(f"面积缩放倍数 = {缩放倍数:.6f}")

        return 缩放倍数

    def 显示信息(self):
        """显示坐标系的信息"""
        print(f"\n坐标系: {self.名称}")
        print(f"维度: {self.维度}")
        print("坐标轴:")
        for i, 轴 in enumerate(self.坐标轴):
            print(f"  轴{i + 1}: {轴}")


def 处理任务组(任务组):
    """
    处理一个任务组
    """
    print("\n" + "=" * 60)
    print(f"开始处理任务组: {任务组['group_name']}")
    print("=" * 60)

    # 创建初始坐标系
    当前坐标系 = 坐标系(任务组['ori_axis'], f"{任务组['group_name']}_初始坐标系")
    当前坐标系.显示信息()

    # 添加所有向量
    print(f"\n共有 {len(任务组['vectors'])} 个向量")

    所有结果 = []

    # 依次执行任务
    for 任务序号, 任务 in enumerate(任务组['tasks']):
        print(f"\n{'-' * 40}")
        print(f"执行任务 {任务序号 + 1}: {任务['type']}")
        print(f"{'-' * 40}")

        结果 = {
            '任务序号': 任务序号 + 1,
            '类型': 任务['type'],
            '成功': False,
            '所有向量的结果': []
        }

        try:
            if 任务['type'] == 'axis_projection':
                # 投影任务：对每个向量计算投影
                for 向量 in 任务组['vectors']:
                    投影结果 = 当前坐标系.投影长度(向量)
                    结果['所有向量的结果'].append({
                        '向量': 向量,
                        '投影长度': 投影结果
                    })
                结果['成功'] = True

            elif 任务['type'] == 'axis_angle':
                # 夹角任务：对每个向量计算夹角
                for 向量 in 任务组['vectors']:
                    夹角结果 = 当前坐标系.夹角弧度(向量)
                    结果['所有向量的结果'].append({
                        '向量': 向量,
                        '夹角弧度': 夹角结果,
                        '夹角角度': [math.degrees(a) for a in 夹角结果]
                    })
                结果['成功'] = True

            elif 任务['type'] == 'area':
                # 面积任务：只计算一次面积缩放
                面积结果 = 当前坐标系.面积缩放()
                结果['所有向量的结果'].append({
                    '面积缩放倍数': 面积结果
                })
                结果['成功'] = True

            elif 任务['type'] == 'change_axis':
                # 坐标系转移任务：创建新坐标系，转移所有向量
                print(f"目标坐标轴: {任务['obj_axis']}")

                # 创建目标坐标系
                目标坐标系 = 坐标系(任务['obj_axis'], f"{任务组['group_name']}_目标坐标系_{任务序号 + 1}")

                # 对每个向量进行转移
                for 向量 in 任务组['vectors']:
                    转移结果 = 当前坐标系.坐标系转移(向量, 目标坐标系)
                    结果['所有向量的结果'].append({
                        '原向量': 向量,
                        '转移结果': 转移结果.tolist()
                    })

                # 更新当前坐标系为目标坐标系
                当前坐标系 = 目标坐标系
                print(f"\n当前坐标系已切换到新坐标系")
                当前坐标系.显示信息()
                结果['成功'] = True

            else:
                print(f"未知的任务类型: {任务['type']}")
                结果['成功'] = False

            所有结果.append(结果)

        except Exception as e:
            print(f"任务执行出错: {e}")
            结果['成功'] = False
            结果['错误信息'] = str(e)
            所有结果.append(结果)

    return 所有结果


def 处理json文件(文件路径):

    读取并处理JSON文件中的所有任务组
    """

        if not os.path.exists(文件路径):

            return

        print(f"文件包含 {len(data)} 个任务组")

            print(f"  共 {len(结果列表)} 个任务，成功 {成功数} 个")

    except FileNotFoundError:
        print(f"找不到文件: {文件路径}")
    except json.JSONDecodeError as e:
        print(f"JSON文件格式错误: {e}")
    except Exception as e:
        print(f"处理过程中出错: {e}")
```
